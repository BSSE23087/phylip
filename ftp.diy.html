Doing the ftp transfer yourself

Normally, the documentation, source code and executables of PHYLIP are downloaded using the web page Get Me PHYLIP and the links in it. If that cannot be done for some reason, one can also get these materials by using ftp yourself.

The programs, including executables, source code and documentation, are available from our server evolution.genetics.washington.edu in directory pub/phylip. A set of detailed instructions on how to get them by anonymous ftp file transfer is available. You can display these instructions on your screen now, or transfer them by ftp as a text file called "Read.Me" which can be read in a word processor program.

  • If nothing happened using the above links, try this one
  • Transfer them as file Read.Me
